**Status**: Draft
**Lead**: Fiona (Product Strategy)

---

## 1. Product Vision
**AetherSynth** is a browser-based generative audio-visual environment. It allows users to interact with a high-fidelity visual Abyss/Aurora system using their microphone and keyboard.

## 2. Core Features
1.  **Reactive Visuals**: A Three.js-based environment that reacts to audio frequencies (Bass/Mids/Treble).
2.  **Interactive Ripples**: Click or keypress events create ripples in the generative geometry.
3.  **Bioluminescent Aesthetic**: A deep, bioluminescent color palette (Violets, Cyans, Emeralds).

## 3. Technology Stack
-   **Core**: Three.js (WebGL)
-   **Audio**: Web Audio API
-   **Animation**: GSAP (GreenSock)
-   **Build Tool**: Vite
-   **Language**: TypeScript

## 4. User Interaction
-   **Passive**: The system reacts to microphone input.
-   **Active**: Keyboard and mouse events trigger visual "shocks" or "ripples."

## 5. Development Roadmap
1.  **Phase 1**: Prototype Audio Engine + Basic Shaders (Complete).
2.  **Phase 2**: Refine Shader aesthetics (Aurora/Abyss balance).
3.  **Phase 3**: User Interface (HUD) and Control Panel.

---

**Technical Goal**: Achieve 60FPS generative visuals on a standard modern laptop.
